\doxysubsection{Detailed Description}
Represents a algorithm to determine the shortest path between two vertices of a graph. Some minor assumptions/modifications\+: we are making an directed/undirected graph into a undirected graph to analyze because we are assuming no one is traveling back and forth between two same airports and there will normally be a flight going back and forth between two places
